Tentang Game

Catch the Falling Objects adalah game sederhana berbasis web. Kamu menggerakkan keranjang untuk menangkap objek yang jatuh dari atas. Dibuat dengan HTML5 Canvas dan JavaScript.

1. Struktur Dasar HTML

Elemen utama pada HTML terdiri dari:

  • <canvas> - Area permainan.
  • <div id="score"> - Menampilkan skor pemain.
  • <div id="gameOverScreen"> - Muncul saat permainan berakhir.
  • <button id="restartBtn"> - Untuk mengulang permainan.

2. CSS untuk Tampilan

CSS digunakan untuk mempercantik tampilan latar, tombol, dan area canvas.

3. JavaScript (Logika Permainan)

a. Inisialisasi

Mendefinisikan canvas, konteks gambar, dan variabel penting seperti posisi pemain, skor, dan objek jatuh.

const canvas = document.getElementById('gameCanvas');
  const ctx = canvas.getContext('2d');

b. Fungsi resetGame()

Mengatur ulang permainan dari awal: skor, posisi, dan memulai loop permainan.

c. Fungsi spawnObject()

Menambahkan objek baru secara acak di atas canvas.

d. Fungsi drawPlayer()

Menggambar pemain (basket) di bagian bawah layar.

e. Fungsi drawObjects()

Menggambar objek bulat berwarna kuning yang jatuh dari atas ke bawah.

f. Fungsi update()

Menggerakkan objek, memeriksa tabrakan antara objek dan pemain, serta mendeteksi game over.

g. Fungsi gameLoop()

Loop utama game menggunakan requestAnimationFrame() untuk memperbarui tampilan.

h. Event Listener Keyboard

Untuk menggerakkan pemain ke kiri atau kanan menggunakan tombol panah:

window.addEventListener('keydown', e => { ... });

4. Penutup

Game ini cocok untuk pembelajaran dasar animasi dan interaksi menggunakan canvas dan JavaScript. Anda dapat menambahkan fitur seperti level, nyawa, atau efek suara untuk membuat game lebih seru!

Selamat mencoba dan bereksperimen dengan kode!

Langkah-langkah Bermain

  1. Gunakan tombol ← (kiri) dan → (kanan) untuk menggerakkan keranjang.
  2. Tangkap objek kuning yang jatuh dari atas layar.
  3. Setiap objek yang berhasil ditangkap menambah skor.
  4. Jika kamu melewatkan satu objek, permainan selesai.
  5. Klik tombol Restart Game untuk mencoba lagi.
